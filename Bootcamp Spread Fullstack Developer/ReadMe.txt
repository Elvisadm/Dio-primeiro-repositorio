	
	Spread Fullstack Developer
	Básico 98hrs 13.04.22 á 17.07.22

Descrição:O Bootcamp spread Fullstack Developer é um programa 
gratuito para pessoas desenvolvedoras que querem começar 
sua jornada em desenvolvimento de software com uma base 
sólida em tecnologias essenciais em front-end quanto 
e back-end.

	A bootcamp está dividido em:
-> Módulo I - Conteúdo de abertura
-> Módulo II - HTML/CSS3
-> Módulo III - JavaScript
-> Módulo IV - ReactJS
-> Módulo V - Banco de Dados
-> Módulo VI -> NodeJS

	No módulo I - Conteúdo de abertura, temos as 3 aulas, 1 live e um projeto, sendo:
* Aula - Boas-vindas ao Bootcamp: Spread Fullstack.
* Live - Mentoria #1: Aula inaugural - spread
* Aula - Lógica de Programação Essencial
* Aula - Introdução ao Git e ao GitHub
* Projeto - Criando seu primeiro Repositório no GitHub 
para Compartilhar seu progresso.

	No módulo II - HTML5/CSS3, temos 3 aulas e 1 live, sendo:
* Aula - Primeiro passos para desenvolvimento web
* Aula - Introdução a criação de websites com HTML5 e CSS3
* Aula - Posicionando elementos com Flexbox em CSS
* Desafio - Recriando a página inicial do Instagram

	No modulo III - JavaScript, temos 12 aulas,1 desafio de codigo 
e 1 desafio de projetos, sendo:
* Aula - Introdução ao javaScript
* Aula - Sintaxe e Operadores
* Aula - Variáveis e Tipos
* Aula - Funções
* Aula - Coleções
* Aula - Map, filter e Reduce
* Aula - Debugging e Error Handling
* Aula - Javascript Assíncrono
* Aula - Orientação a objetos
* Aula - Trabalhando com Módulos em JavaScript
* Aula - Manipulando a D.O.M com JavaScript
* Aula - Introdução ao TypeScript: Explorando Classes, Tipos e Interfaces
* Código - Desafios Iniciais Js - Spread FullStack
* Projeto - Aprenda a criar um sistema de estacionamento usando TypeScript

	No Módulo IV - ReactJS, temos 6 aulas e 1 desafio de projeto e 1 de codigo, sendo:
* Aula - Introdução ao reactJs
* Aula - Introdução aos React Hooks
* Aula - Trabalhando com States & Effects no Reactjs
* Aula - Desenvolvimento de aplicações para internet com ReactJS
* Desafio - Intermediários Js - Spread FullStack
* Aula - Introdução a redux com ReactJs
* Projeto - Criando um front-end totalmente componentizado na prática
com reactJs

	No Modulo v - Banco de Dados, temos 2 aulas sendo:
* Aula - MySQL - Trabalhando com as suas primeiras tabelas
* Aula - MySQL - Consultas com Join

	No Módulo VI - NodeJS, temos 5 aulas, 1 desafio de projetos e 1 de codigo, sendo:
* Aula - Introdução a APIs e métodos HTTP
* Aula - Desenvolvimento back-end com Node.JS
* Aula - Introdução ao Node.JS com Express
* Código - spread FullStack
* Aula - Explorando o Estilo Arquitetural REST com Node.js
* Aula - Node.js com Bancos de Dados Relacionais (SQL)
* Projeto - Contruindo Encurtador de URl.
